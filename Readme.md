# CrewAI-AgenticTools

A collection of minimal, practical examples demonstrating how to build and use tools with CrewAI.

This repository focuses on understanding how different tools can be integrated into agent workflows using simple, isolated implementations.

---

## Overview

This project contains multiple Jupyter Notebook (`.ipynb`) files, each showcasing the implementation and usage of a specific tool. The goal is to provide clear, minimal examples for learning how tools work within CrewAI.

These examples are intentionally lightweight and focus on syntax, structure, and basic functionality rather than complex workflows.

---

## Tools Included

### 1. Custom Tool – `textLengthCalculator` (BaseModel + BaseTool)
- A simple custom tool that calculates the length of a given text input
- Demonstrates how to create a custom tool using CrewAI’s base classes
- Uses `BaseModel` for input schema validation
- Uses `BaseTool` for defining tool behavior
- Helps understand how to extend CrewAI with your own logic

---

### 2. DALL·E Tool
- Example of generating images using DALL·E
- Shows how agents can interact with image generation APIs
- Useful for multimodal agent workflows

---

### 3. Serper Tool
- Demonstrates integration with Serper.dev for web search
- Enables agents to fetch real-time search results
- Useful for research-based tasks

---

### 4. File Read & Write Tool
- Reads content from files
- Writes output generated by agents to files
- Demonstrates basic persistence and I/O handling

---

### 5. PDF Search Tool
- Extracts and searches content within PDF documents
- Useful for document-based querying and summarization tasks

---

### 6. Web Scraping Tool
- Fetches and processes data from web pages
- Demonstrates how agents can gather external data dynamically

## Getting Started

### 1. Clone the Repository
```
git clone https://github.com/your-username/CrewAI-AgenticTools.git
cd CrewAI-AgenticTools
```
### 2. Install Dependencies

Make sure you have Python 3.9+ installed, then install required packages:
```
pip install crewai
pip install openai
pip install requests
pip install beautifulsoup4
pip install pypdf
```
(Adjust dependencies based on your implementation.)

---

### 3. Set Environment Variables

Some tools require API keys:

- OpenAI API key (for DALL·E)
- Serper.dev API key

Set them in your environment:
```
export OPENAI_API_KEY=your_key_here
export SERPER_API_KEY=your_key_here
```
---

### 4. Run the Notebooks

Open Jupyter Notebook or VS Code and run each `.ipynb` file to see tool usage in action.
